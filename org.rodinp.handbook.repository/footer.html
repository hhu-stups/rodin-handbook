Installing the Rodin Handbook plug-in

Please use the Rodin update manager to install the plug-ins.

  1. Select from main menu: Help | Install New Software...
  2. Under Work with, select the Rodin Handbook repository. If it isn't listed, add a new repository using the URL
    https://stups.hhu-hosting.de/handbook/rodin/updatesite/
  3. Select the Rodin Handbook plug-in and follow the dialog.
